# CSCI-282-Assignment4
## Brandon Terry
### April 10th 2026

This project is the implementation of a compiler for a small language called Diamondback, which supports 32-bit integers, booleans, local variables, function definitions and calls, recursion, and several unary and binary operations. It includes all the source code and a test file that contains at least 25 test cases covering function calls, multiple arguments, recursion, mutual recursion, and variable access inside functions.
